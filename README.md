# dracula-css
The colours from the [Dracula Theme](https://draculatheme.com/) as a set of css variables.


```css
:root {
  --background: #282a36;
  --current-line: #44475a;
  --foreground: #f8f8f2;
  --comment: #6272a4;
  --cyan: #8be9fd;
  --green: #50fa7b;
  --orange: #ffb86c;
  --pink: #ff79c6;
  --purlple: #bd93f9;
  --red: #ff5555;
  --yellow: #f1fa8c;
}
```


You may also want to make use of:
```css
:root {
  --background-lighter: #383a59;
}
```
